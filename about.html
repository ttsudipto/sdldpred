SDLDpred - Symptom-based Drugs of Lifestyle-related Diseases prediction

Home About Help Team
  • SDLDpred is a web-based tool for prediction of drugs of lifestyle-related diseases from symptom data.
  • It used unsupervised machine learning models trained with Bisecting K-Means algorithm using drug-symptom association data of 143 lifestyle-related diseases, with focus on pulmonary and cardiovascular diseases. A summary of the diseases are given in the following table.

    Disease category Number of diseases
    Pulmonary diseases 67
    Cardiovascular diseases 56
    Musculoskeletal diseases 15
    Diabetes 4
    Obesity 1
    Total 143

  • A set of drug-symptom associations were computed based on cosine similarity between each drug and symptom represented as a vector of disease-associations.
    dri = ith drug represented as a vector of associations with the diseases, sj = jth symptom represented as a vector of associations with the diseases, ||x|| = Magnitude of the vector x.

    The drug-symptom association dataset: Download [Matrix (TSV file)]; Download [Edge-list (TSV file)]
  • SDLDpred provides a confidence score for each predicted drug. It is computed using the following equation.
    D = random variable denoting the pairwise euclidean distances between the drugs; e = euclidean distance of a predicted drug; fdist = probability density function of the random variable D, fdist (x) = Prob (D = x).
